# II - Formulaire

> [!tips]
> $\cos$ est un connard, $\sin$ est sympa.


## 1. Trigonométrie 
- **Additivité**
	- $\cos(a+b) = \cos a \cos b - \sin a \sin b$
	- $\cos(a-b) = \cos a \cos b + \sin a \sin b$
	- $\sin(a+b) = \sin a \cos b + \sin b \cos a$
	- $\sin(a-b) = \sin a \cos b - \sin b \cos a$
- **Identité**
	- $\sin^2x+\cos^2x=1$

### 1.1 Exemples de corollaires

- $\cos(2x) = 2\cos^2x-1 = \cos^2x-\sin^2x$
- $\sin(2x) = 2\sin x\cos x$
